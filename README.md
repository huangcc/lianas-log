# Liana's Log

A baby tracking app for logging feeds, diapers, and sleep — built for hands-free voice input via Siri Shortcuts.

## What it does
- Voice logging via Siri Shortcuts → Google Apps Script webhook
- Live nursing timer and feed/diaper charts
- Claude API-powered natural language logging
- Advice tab (in progress)

## How it's built
- **Frontend:** GitHub Pages (plain HTML/CSS/JS)
- **Backend:** Google Apps Script
- **Data store:** Google Sheets
